Tools and Technology

A standard laptop and an IOS are the minimum requirements for the development of the application. Microsoft Teams will be essential for colleagues to communicate through and meet. In the design stage of the application, Figma will be used to create wireframes and a prototype of how the application would look and function. Furthermore, the tool that will be used for creating the application is Xamarin. As Xamarin is an open source platform a software license would not be required.

The platform Microsoft Azure will be required to manage data and services, as well as being able to update the app regularly when necessary. This will be useful as the project will require large amounts of stored information that will most likely expand if the number of users increase. To allow users to register accounts or login using social media, OAuth framework will be used to manage logins and ensure a high level of security.

Although none of the group members have any experience in using the application development tools and platforms, everyone in the group possesses fundamental knowledge of Figma and Microsoft Teams, which can be used to design a general prototype of how the application would look and the features that will be included within the application and its functions.
